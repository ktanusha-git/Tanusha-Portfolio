## How to View (No Coding Required)

### Option 1: Open on Your Computer
1. Find the file **`index.html`** in this folder
2. **Double-click** it
3. It will open in your web browser

### Option 2: Share Online (Free with GitHub)
Get a link like `https://yourusername.github.io/Tanusha-Portfolio/` that anyone can open.

**Step-by-step:**

1. **Create a GitHub account** (free) at [github.com](https://github.com)
2. **Create a new repository:**
   - Click the **+** in the top-right → **New repository**
   - Name it: `Tanusha-Portfolio` (or any name you like)
   - Leave everything else as default → **Create repository**
3. **Upload your files:**
   - Click **"uploading an existing file"**
   - Drag and drop: `index.html` (and any other files in this folder)
   - Click **Commit changes**
4. **Turn on GitHub Pages:**
   - Go to **Settings** (top menu of your repo)
   - Click **Pages** in the left sidebar
   - Under "Source", select **Deploy from a branch**
   - Under "Branch", select **main** and **/ (root)**
   - Click **Save**
5. **Wait 1–2 minutes**, then visit:  
   `https://YOUR-GITHUB-USERNAME.github.io/Tanusha-Portfolio/`

Replace `YOUR-GITHUB-USERNAME` with your actual GitHub username.
